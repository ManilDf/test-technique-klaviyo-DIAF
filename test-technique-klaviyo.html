01 — Contexte

Présentation du projet

Vous intégrez une équipe qui développe une plateforme de fidélité client. Cette plateforme synchronise les données clients en temps réel vers Klaviyo, un outil d'email marketing et d'automatisation.

L'architecture repose sur un pattern répété à chaque entité : un mapper lit les données depuis SQL Server via ADO.NET et construit un payload C#, puis un service de synchronisation envoie ce payload à l'API Klaviyo. L'injection de dépendances est assurée par ASP.NET Core.

Règle fondamentale : Aucun ORM (Entity Framework, Dapper…) n'est autorisé. Toutes les requêtes SQL doivent être en ADO.NET pur, écrites inline dans le code C#. Aucune procédure stockée.
02 — Objectif

Ce que vous devez construire

Vous devez implémenter les trois composants nécessaires à la synchronisation d'un membre du programme fidélité (Member) vers un profil Klaviyo :

KlaviyoMember.cs — Modèle C# représentant le payload JSON Klaviyo. Toutes les propriétés doivent porter l'attribut [JsonPropertyName]. Les classes imbriquées (location, properties) sont à inclure dans ce fichier.
IKlaviyoMemberMapper + KlaviyoMemberMapper — Interface et implémentation. Le mapper lit les tables Member et MemberEmail via ADO.NET et retourne un objet KlaviyoMember prêt à l'envoi.
IKlaviyoMemberSyncService + KlaviyoMemberSyncService — Interface et implémentation. Le service orchestre l'appel au mapper puis l'envoi à l'API Klaviyo via IKlaviyoApiClient.

04 — Base de données

Schéma des tables

schema.sql 
CREATE TABLE [dbo].[Member] (
    MemberId           INT            NOT NULL PRIMARY KEY,
    FirstName          NVARCHAR(100)  NOT NULL,
    LastName           NVARCHAR(100)  NOT NULL,
    PhoneNumber        NVARCHAR(20)   NULL,
    BirthDate          DATETIME       NULL,
    City               NVARCHAR(100)  NULL,
    Country            NVARCHAR(100)  NULL,
    LoyaltyPoints      INT            NOT NULL DEFAULT 0,
    MainEmailAddressId INT            NULL,
    EnrolledAt         DATETIME       NOT NULL
);

CREATE TABLE [dbo].[MemberEmail] (
    MemberEmailId INT            NOT NULL PRIMARY KEY,
    MemberId      INT            NOT NULL,
    EmailAddress  NVARCHAR(255)  NOT NULL
);
Les colonnes PhoneNumber, BirthDate, City, Country et MainEmailAddressId peuvent être NULL. Votre mapper doit les gérer sans lever d'exception.
05 — API Klaviyo

Endpoint et payload attendu

L'interface IKlaviyoApiClient fournie gère déjà les headers HTTP. Votre service appelle simplement CreateOrUpdateProfileAsync(payload, accessToken) et vérifie le code retour.

Endpoint
POST https://a.klaviyo.com/api/profiles/

Headers (gérés par KlaviyoApiClient) :
Authorization: Klaviyo-API-Key {accessToken}
revision: 2024-02-15
Content-Type: application/json
Accept: application/json

Réponse succès : 200 ou 201
Réponse erreur : 4xx / 5xx → throw
Corps JSON attendu
{
  "data": {
    "type": "profile",
    "attributes": {
      "email": "jean.dupont@email.com",
      "first_name": "Jean",
      "last_name": "Dupont",
      "phone_number": "+33612345678",
      "birthday": "1985-03-21",
      "location": {
        "city": "Paris",
        "country": "France"
      },
      "properties": {
        "loyalty_points": 450,
        "enrolled_at": "2024-01-15T10:30:00Z"
      }
    }
  }
}

Format birthday : yyyy-MM-dd  ·  Format enrolled_at : ISO 8601 UTC (yyyy-MM-ddTHH:mm:ssZ)  ·  phone_number : null si absent en base

07 — Travail demandé

Signatures à respecter

Vos implémentations doivent exposer les interfaces suivantes. La logique interne est laissée à votre appréciation.

Klaviyo/Mappers/IKlaviyoMemberMapper.cs 
namespace KlaviyoTest.Klaviyo.Mappers;

public interface IKlaviyoMemberMapper
{
    /// Lit le membre depuis SQL Server et retourne le payload Klaviyo formaté.
    /// Lève KeyNotFoundException si le membre n'existe pas en base.
    Task<KlaviyoMember> GetMemberAsync(int connectionId, int memberId);
}
Klaviyo/Services/IKlaviyoMemberSyncService.cs 
namespace KlaviyoTest.Klaviyo.Services;

public interface IKlaviyoMemberSyncService
{
    /// Mappe le membre et envoie le profil vers Klaviyo.
    /// Lève InvalidOperationException si la réponse API n'est pas 2xx.
    Task SyncMemberAsync(int connectionId, int memberId, string accessToken);
}
